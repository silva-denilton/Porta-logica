Conectivo Lógico NOT "(¬)"

Porta AND Porta OU Porta IMPLICA Porta BI-IMPLICA

Definição


Uma conjunção é representada pelo conectivo lógico "Not"(¬), A função NOT é aquela que inverte ou complementa o estado da variável de entrada, ou seja, se a variável for False (Q), a saída vai ficar True (P), e se for True (P) a saída vai ficar False (Q). É, portanto, uma porta com apenas um sinal de entrada e um sinal de saída, o qual assumirá sempre valores lógicos inversos.{ 1 }

Exemplos em linguagem natural:


Fórmula (H) = (¬P ^ ¬Q)

P = João gosta de futebol.
Q = Marcelo gosta de vôlei.
H = (¬)João gosta de futebol (E) (¬)Marcelo gosta de vôlei.

Tabela Verdade




P ¬P
true false
false true


Equivalência Semântica


Podemos dizer que duas fórmulas são equivalentes semanticamente, se o resultado de ambas as tabelas-verdade forem iguais. A seguir temos alguns exemplos de algumas fórmulas que equivalem semanticamente à conjunção entre "P" e "Q" (¬P ^ ¬Q).

Exemplo


¬(P ^ Q):

¬(P ^ Q) P Q H
true true true true
true true false true
true false True true
false false false false

Nos exemplos acima podemos ver que os resultados obtidos em "H", são idênticos ao resultado da fórmula ¬(P ^ Q) sendo verdadeiro nas primeira, segunda e terceira interpretações e falso apenas na quarta.
